# 📚 私人题库系统 (Question Bank System)

一个功能完善的个人题库管理系统，支持多学科题目管理、智能练习、考试模拟、闪卡记忆、AI 辅助、学习报告等功能。

## ✨ 功能特性

### 📝 题库管理
- **多学科支持**：语文、数学、英语、历史、物理、化学、生物、地理、政治
- **多种题型**：单选、多选、填空、简答、论述、判断
- **批量导入**：支持文本粘贴导入、Excel (.xlsx/.xls/.csv) 文件导入
- **智能去重**：导入时自动检测重复题目
- **阅读材料**：支持关联阅读材料（阅读理解、文言文等）
- **教材目录**：按教材版本组织题目（鲁教版、人教版等）
- **题目关联**：手动或自动建立题目关联（同知识点、相似题、前置题）
- **题目笔记**：每道题可记录个人理解、易错点、解题思路
- **题目版本历史**：每次编辑自动保存快照，支持查看历史版本
- **全文搜索**：SQLite FTS5 全文索引，搜索更快更准
- **高级搜索**：搜索支持 `subject:数学 difficulty:3` 过滤语法
- **操作历史**：关键操作记录，支持撤销恢复
- **题目图片**：题干、答案、解析均支持插入图片
- **批量操作**：题目列表支持多选 → 批量删除 / 批量修改学科 / 难度 / 分类
- **导入预览**：文本导入前可预览解析结果，显示重复题目
- **Excel 模板下载**：提供标准 Excel 导入模板

### 🎯 练习系统
- **随机练习**：按学科、分类、难度、题型筛选出题
- **自适应难度**：根据答题表现自动调整出题难度（正确率 >85% 升难，<40% 降难）
- **错题本**：自动收录错题，支持 SM-2 间隔重复调度
- **错题本快捷键**：复习模式下 `1` `2` `3` 快速评估掌握程度，`←` `→` 切换题目，`Space` 显示答案
- **闪卡快捷键**：闪卡复习 `1`/`2`/`3`/`4` 评估掌握度，`←`/`→` 切卡
- **收藏夹**：收藏重点题目
- **答题统计**：正确率、答题趋势、学科能力雷达图
- **语音朗读**：Web Speech API 朗读题目内容

### 📄 试卷系统
- **手动组卷**：拖拽排序选题
- **自动组卷**：按题型 + 难度 + 数量规则一键生成试卷
- **打印导出**：支持打印预览，可导出为 PDF（Ctrl+P）
- **考试模拟**：限时考试，自动评分，音频倒计时提醒
- **考试状态持久化**：考试进度自动保存到本地，刷新页面不丢失
- **试卷对比**：同一试卷多次考试的成绩趋势 + 每题得分率分析

### 🃏 闪卡记忆
- **SM-2 算法**：基于遗忘曲线的间隔重复
- **自动调度**：根据掌握程度自动安排复习时间

### 🏆 答题闯关
- **学科选关**：选择学科开始闯关挑战
- **等级递进**：难度随关卡递增，题数逐渐增多
- **连击系统**：连续答对有 🔥 连击提示
- **成就存储**：最高通关记录保存在本地

### 📊 数据分析
- **学科正确率**：各学科答题正确率统计
- **薄弱知识点**：自动识别薄弱环节并推荐练习
- **学习趋势**：每日答题量和正确率趋势图
- **学习报告**：周报/月报，汇总答题数、正确率、错题动态、学科/难度分析
- **学习日历**：90 天 GitHub 风格热力图，直观展示每日学习量
- **Dashboard 增强**：今日答题、本周答题、今日新增、待复习错题等统计卡片

### 🤖 AI 智能
- **AI 导入**：通过 AI 自动解析题目文本，智能识别题型、难度、分类
- **AI 出题**：根据知识点自动生成新题目
- **AI 改写**：将题目改写为不同难度或题型
- **AI 详解**：为题目生成详细的解题步骤和思路分析

### 📤 导出增强
- **Anki 导出**：导出为 Anki TSV 格式，直接导入 Anki 复习
- **错题本 PDF**：导出错题本为可打印的 HTML 页面（Ctrl+P → PDF）
- **错题本 Word**：导出错题本为 `.docx` Word 文档
- **学习报告 PDF**：学习报告导出为可打印 HTML 页面
- **CSV/Word**：题目列表导出为 CSV 或 Word 文档
- **全量备份**：导出完整数据库快照（含题目、试卷、闪卡、笔记、单词等）

### 🔌 实时通知
- **WebSocket**：导入完成等事件实时推送
- **自动刷新**：Dashboard 收到通知后自动更新统计

### 🔒 用户系统
- **多用户支持**：管理员 + 普通用户
- **数据隔离**：每个用户的答题记录、错题本、收藏夹独立
- **JWT 认证**：7天自动过期的安全认证
- **密码修改**：设置页支持修改密码
- **管理后台**：用户管理（增删用户）、审计日志查看
- **审计日志**：记录关键操作（登录、删除等）

### 📱 PWA 离线支持
- **Service Worker**：静态资源和 API 响应缓存
- **离线访问**：断网时可访问已缓存的题目
- **添加到主屏幕**：像原生 App 一样使用

## 🚀 快速开始

### 环境要求
- Node.js >= 18
- npm 或 yarn

### 安装

```bash
# 克隆项目
git clone https://github.com/zhaoyuezheng-cvg/question-bank.git
cd question-bank

# 安装依赖（自动执行 prisma generate）
npm install

# 初始化/同步数据库结构
npm run db:push

# 可选：导入种子数据
npm run db:seed
```

> **注意**：每次 `git pull` 拉取新代码后，如果 `schema.prisma` 有变更，需重新运行 `npm run db:push` 同步数据库。

### 启动

```bash
# 同时启动前后端
npm run dev

# 或分别启动
npm run dev:backend   # 后端，默认端口 3000
npm run dev:frontend  # 前端，默认端口 5173
```

打开浏览器访问 `http://localhost:5173`，首次使用会引导创建管理员账号。

### 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `PORT` | 后端端口 | `3000` |
| `JWT_SECRET` | JWT 密钥（生产环境务必修改） | `question-bank-secret-key-change-in-production` |
| `AI_API_KEY` | AI API Key（可选，也可在页面配置） | - |
| `AI_API_BASE` | AI API 地址（可选） | `https://api.openai.com/v1` |
| `AI_MODEL` | AI 模型名称（可选） | `gpt-4o-mini` |
| `DATABASE_URL` | 数据库路径（可选） | `file:./data/question-bank.db` |

## 🐳 Docker 部署

### 使用 Docker Compose（推荐）

```bash
# 1. 克隆仓库
git clone https://github.com/zhaoyuezheng-cvg/question-bank.git
cd question-bank

# 2. 创建 .env 文件（可选）
cp .env.example .env
# 编辑 .env 设置 JWT_SECRET 和 AI_API_KEY

# 3. 启动
docker-compose up -d

# 4. 查看日志
docker-compose logs -f
```

启动后访问 `http://localhost:3000`

### 使用 Docker

```bash
# 构建镜像
docker build -t question-bank .

# 运行容器
docker run -d \
  --name question-bank \
  -p 3000:3000 \
  -v qb-data:/app/data \
  -v qb-uploads:/app/uploads \
  -e JWT_SECRET=your-secret-here \
  question-bank
```

## 📖 使用指南

### 导入题目

#### 文本导入
1. 进入「批量导入」页面
2. 选择默认学科、分类
3. 粘贴包含标记的文本，支持三种格式：
   - **标记格式**：`【题干】...【答案】...【解析】...`
   - **编号格式**：`1. 题目内容`
   - **空行分隔**：多题之间用空行隔开
4. 点击「开始导入」

#### Excel 导入
1. 准备 Excel 文件，表头支持中英文：
   - 学科/subject、题型/type、分类/category、难度/difficulty
   - 题干/content、选项/options、答案/answer、解析/analysis
   - 标签/tags、来源/source
2. 上传文件，点击「开始导入」
3. 系统自动去重，跳过已存在的题目

### 插入图片
在新建/编辑题目页面，题干、答案、解析三个区域都有「🖼️ 插入图片」按钮，点击后选择图片上传，系统自动插入 Markdown 图片语法。

### 自适应练习
1. 进入「答题练习」页面
2. 勾选「🧠 自适应难度」
3. 系统根据你近 7 天的答题表现自动推荐难度：
   - 正确率 ≥ 85%：推荐难度 4（高难度 4 级正确率也高则推荐 5）
   - 正确率 60%-84%：推荐难度 3（保持当前）
   - 正确率 40%-59%：推荐难度 2（降难巩固）
   - 正确率 < 40%：推荐难度 1（从基础开始）

### 自动组卷
1. 进入「试卷管理」页面
2. 点击「🤖 自动组卷」
3. 选择学科，设置规则（题型 + 难度 + 数量）
4. 点击「生成试卷」

### 错题间隔复习
1. 进入「错题本」页面
2. 点击「🔄 间隔复习」进入复习模式
3. 查看题目后选择掌握程度：
   - 😵 完全忘了 → 1 天后重练（快捷键 `1`）
   - 😐 有印象 → 按 SM-2 算法调度（快捷键 `2`）
   - 😊 记得住 → 延长间隔，连续 3 次「记住得」自动标记为已掌握（快捷键 `3`）
4. 复习模式下可用 `←` `→` 切换题目，`Space` 显示答案

### 学习报告
1. 进入「学习报告」页面
2. 选择时间范围（7 天 / 14 天 / 30 天）
3. 查看：总答题数、学习天数、正确率、日均答题、错题动态、学科表现、难度分析、每日趋势
4. 点击「📄 导出 PDF」可打印为 PDF 文件

### Dashboard 仪表盘
- **统计卡片**：题目总数、试卷数量、已答题数、正确率
- **扩展统计**：今日答题、本周答题、今日新增、待复习错题
- **学科分布**：饼图展示各学科题目占比
- **难度分布**：柱状图展示各难度等级题目数
- **题型分布**：柱状图展示各题型数量
- **学习日历**：90 天 GitHub 风格热力图，颜色越深表示当日答题越多
- **复习提醒**：顶部横幅显示待复习闪卡和错题数量

### 导出
- **Anki**：题目管理页 → 📥 Anki → 下载 TSV 文件 → 导入 Anki
- **错题本 PDF**：错题本 → 📄 导出 PDF → Ctrl+P 打印为 PDF
- **错题本 Word**：错题本 → 📝 导出 Word → 下载 `.docx` 文件
- **学习报告 PDF**：学习报告 → 📄 导出 PDF → Ctrl+P 打印为 PDF
- **全量备份**：侧边栏 → 💾 备份数据 → 下载 JSON 文件
- **恢复备份**：侧边栏 → 📥 恢复备份 → 选择 JSON 文件

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `N` | 新建题目 |
| `⌘K` / `Ctrl+K` | 打开搜索面板（支持高级搜索语法） |
| `?` | 显示快捷键帮助 |
| `Esc` | 关闭弹窗 |
| `A` `B` `C` `D` | 练习模式选择答案 |
| `Enter` | 提交答案 / 下一题 |
| `Space` | 错题/闪卡复习显示答案 |
| `1` `2` `3` | 错题复习评估掌握程度 |
| `1` `2` `3` `4` | 闪卡复习评估掌握程度 |
| `←` `→` | 错题/闪卡复习切换题目 |
