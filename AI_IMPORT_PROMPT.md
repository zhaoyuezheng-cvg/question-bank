# 🤖 外部 AI 试题解析提示词

将以下提示词发送给通义千问、豆包、DeepSeek 等 AI 助手，附上你的试题文本，即可生成可直接导入的 JSON 数据。

---

## 提示词（直接复制使用）

```
你是一个专业的试题解析助手。请将我提供的试题文本解析为结构化的 JSON 数组格式。

输出格式要求：

```json
[
  {
    "subject": "chinese",
    "category": "现代文阅读",
    "subCategory": "散文阅读",
    "type": "short_answer",
    "subType": "赏析句子",
    "difficulty": 3,
    "content": "题目内容（完整题干）",
    "options": null,
    "answer": "标准答案",
    "analysis": "解题解析",
    "tags": ["标签1", "标签2"],
    "source": "来源"
  }
]
```

字段说明：

subject（学科）：chinese / math / english / history / physics / chemistry / biology / geography / politics

category（一级分类）：
- 语文：现代文阅读 / 古诗文阅读 / 名著阅读 / 语言文字运用 / 基础知识
- 英语：阅读理解 / 完形填空 / 任务型阅读 / 语法填空 / 写作
- 数学：选择题 / 填空题 / 解答题 / 阅读理解题
- 其他学科：选择题 / 材料分析题 / 实验探究题 / 简答题 / 论述题

subCategory（二级分类）：
- 现代文阅读下：小说阅读 / 散文阅读 / 说明文阅读 / 议论文阅读 / 非连续性文本阅读
- 古诗文阅读下：文言文阅读 / 古诗词鉴赏
- 英语阅读理解下：细节理解 / 主旨大意 / 推理判断 / 词义猜测

type（题型）：choice / multi_choice / fill_blank / short_answer / essay / true_false

subType（细分题型，阅读理解专用）：
- 语文：概括内容 / 赏析句子 / 分析标题 / 分析人物 / 环境描写 / 情节梳理 / 主旨探究 / 语言赏析 / 情感体悟 / 词语理解 / 开放题 / 实词虚词 / 翻译句子 / 断句 / 文言理解 / 意象分析 / 手法赏析 / 古诗情感
- 英语：Detail / MainIdea / Inference / Vocabulary / Reference / Attitude / Title
- 通用：实验探究 / 材料分析 / 图表分析 / 论述题

difficulty（难度 1-5）：1=基础 2=较易 3=中等 4=较难 5=困难

options（选项）：选择题时为字符串数组 ["选项A文本", "选项B文本", ...]，不要包含 A/B/C/D 字母前缀。其他题型为 null。

answer（答案）：选择题填字母如 "A"，多选题如 "ABD"，简答题填完整答案。

analysis（解析）：详细的解题思路和解析。

tags（标签）：从题目中提取的关键词数组。

source（来源）：如 "2024全国甲卷"、"人教版七年级" 等，未知则留空。

重要规则：
1. 严格输出 JSON 数组，不要输出其他任何内容
2. 如果原文没有明确答案或解析，对应字段留空字符串 ""
3. 阅读理解一篇文章下有多道题时，每道题独立解析为一个对象
4. 保持 content 中的原始格式（如填空横线 ___）
5. 如果无法判断学科，默认 chinese

请解析以下试题：
（在此粘贴你的试题文本）
```

---

## 使用方法

1. 复制上面的提示词
2. 发送给 AI 助手（千问/豆包/DeepSeek 等）
3. 在提示词末尾粘贴你的试题文本
4. AI 会返回 JSON 格式的结构化数据
5. 在题库系统中：进入「批量导入」页面
6. 粘贴 AI 返回的 JSON 到文本框
7. 点击「开始导入」
