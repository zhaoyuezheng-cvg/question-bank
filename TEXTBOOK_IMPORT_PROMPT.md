# 📚 教材目录导入提示词

将以下提示词发送给通义千问、豆包、DeepSeek 等 AI，附上教材目录信息，即可生成可直接导入的 JSON 数据。

---

## 提示词（直接复制使用）

```
请将以下教材目录信息解析为 JSON 格式，用于导入题库系统。

输出格式：

```json
[
  {
    "name": "教材名称（如：鲁教版数学八年级上册）",
    "subject": "学科代码",
    "grade": "年级",
    "version": "版本（如：鲁教版/人教版）",
    "volume": "册次（如：上册/下册）",
    "chapters": [
      {
        "name": "第一章 三角形",
        "sections": ["1.1 认识三角形", "1.2 图形的全等", "1.3 探索三角形全等的条件"]
      },
      {
        "name": "第二章 轴对称",
        "sections": ["2.1 轴对称现象", "2.2 探索轴对称的性质"]
      }
    ]
  }
]
```

学科代码对照：
- math = 数学
- english = 英语
- chinese = 语文
- physics = 物理
- chemistry = 化学
- biology = 生物
- history = 历史
- politics = 道德与法治
- geography = 地理

规则：
1. 严格输出 JSON 数组，不要输出其他内容
2. 每个 chapter 的 sections 是字符串数组
3. sections 可以用「序号+名称」格式，也可以只用名称
4. 如果目录有上下册，分成两个对象（volume 分别填"上册"和"下册"）
5. 如果是英语，按 Unit 组织章节

请解析以下教材目录：
（在此粘贴教材目录信息）
```

---

## 使用方法

1. 复制上面的提示词
2. 发给 AI 助手（千问/豆包/DeepSeek 等）
3. 在末尾粘贴教材目录（可以是图片OCR文字、网页复制、PDF文字等）
4. AI 返回 JSON 数据
5. 在题库系统中：进入「📚 教材目录」页面
6. 点击「📥 导入目录」
7. 粘贴 JSON 数据，点击「开始导入」

**输入（给 AI 的文本）：**
```
鲁教版数学八年级上册目录：
第一章 全等三角形

  1.2 怎样判定三角形全等

  3.2 分式的约分
  3.3 分式的乘除法
  3.4 分式的加减法
  3.5 分式方程
```

```json
[
  {

    "subject": "math",
    "grade": "八年级",

      {
        "name": "第一章 全等三角形",
        "sections": ["全等三角形", "怎样判定三角形全等", "尺规作图"]
      },
      {
        "name": "第二章 图形的轴对称",
        "sections": ["图形的轴对称", "轴对称的基本性质", "轴对称与坐标变化"]
      },

  }

```

## 批量导入多个教材

如果要一次性导入多个教材（如数学+英语），让 AI 返回包含多个对象的数组：

```json
[
  { "name": "鲁教版数学八年级上册", "subject": "math", ... },
  { "name": "鲁教版英语八年级上册", "subject": "english", ... }
]
```

系统会自动处理：已存在的目录会更新，不存在的会新建。
